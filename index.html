Памятка для учеников

Для 1-4 класса

При организации работы в начальной школе необходимо учитывать возрастные психолого-физиологические особенности детей младшего школьного возраста.

  • Темы детских работ выбираются из содержания учебных предметов
  • Проблема проекта должна быть в области познавательных интересов ребёнка
  • Длительность выполнения проекта — 3-4 недели
  • Текстовый файл не обязателен!

Для 5-7 класса

Проектная деятельность организуется в групповых формах. Темы выбираются из любой содержательной области. Получаемый результат должен быть социально и практически значимым.

Для 8-10 класса

Формирование надлежащего уровня компетентности. Темы подбираются в соответствии с личностными предпочтениями. Предпочтительны индивидуальные формы работы.

Схема проекта

1. Поиск проблемы — Постановка проблемы ключевой этап.

2. Тематизация — Тема фиксирует суть проекта.

3. Цель проекта — Цель формулируется кратко.

4. Планирование — Мини-план работы.

5. Задачи — Задачи определяют как достичь цели.

6. Теоретическая часть — Работа с информацией.

7. Практическая часть — Предложение решения.

8. Заключение — Анализ полученных данных.

Виды проекта

1. Исследовательский — Отвечает на вопрос о том как устроен мир. Включает гипотезу, эксперимент, анализ результатов.

2. Творческий — Максимально свободный подход. Альманахи, театрализации, видеофильмы.

3. Социо-гуманитарный — Поиск решения социальных проблем.

4. Инженерный — Доведение проекта до практического результата.

Площадки для поиска тем

  • Проектория — отечественные компании
  • Globallab — для исследователей
  • Школа реальных дел — кейсы от компаний
  • Школа IT-решений — IT-кейсы
  • Мосгортех — московские предприятия
  • Энергопрорыв — энергетика

Оформление и содержание проекта

Титульный лист: Наименование учебного заведения, тема проекта, ФИО автора и руководителя, город и год.

Паспорт: Тема, ФИО, график работы.

Оглавление: Наименование глав с номерами страниц.

Введение (1-2 стр): Проблема, актуальность, цель, задачи.

Теоретическая часть (7-10 стр): Подходы к исследованию.

Практическая часть: 4-6 кл: 4-7 стр, 7-8 кл: 6-8 стр, 10 кл: 10-15 стр.

Заключение (1-2 стр): Вывод результатов.

Список литературы: Источники информации.

Важно! Работа в 10 классе проверяется на антиплагиат (оригинальность >60%).

Оформление текста

  • Редактор: Microsoft Word
  • Формат: А4, белая бумага
  • Поля: 30/20/15/20 мм
  • Шрифт: Times New Roman 14pt
  • Интервал: полуторный
  • Нумерация: сквозная
  • Титульный лист — без номера

Таблицы: Пустые ячейки — "-" или "нет данных".

Оформление презентации

Электронная презентация — набор слайдов для демонстрации.

  • Минимум текста
  • Текст не визуальное средство
  • Не размещать много на одном слайде
