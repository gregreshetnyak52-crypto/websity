Памятка

1-4 класс

Учитывать возрастные особенности детей младшего школьного возраста.

  • Темы из содержания учебных предметов
  • Проблема в области познавательных интересов
  • Длительность — 3-4 недели
  • Текстовый файл не обязателен!

5-7 класс

Групповые формы работы. Темы из любой области. Результат должен быть социально значимым.

8-10 класс

Формирование компетентности. Индивидуальные формы работы.

Схема проекта

1. Поиск проблемы — Постановка проблемы.

2. Тематизация — Суть проекта.

3. Цель — Краткая формулировка.

4. Планирование — Мини-план.

5. Задачи — Как достичь цели.

6. Теория — Работа с информацией.

7. Практика — Предложение решения.

8. Заключение — Анализ данных.

Виды проекта

1. Исследовательский — Гипотеза, эксперимент, анализ.

2. Творческий — Свободный подход.

3. Социо-гуманитарный — Социальные проблемы.

4. Инженерный — Практический результат.

Площадки для поиска

  • Проектория — отечественные компании
  • Globallab — для исследователей
  • Школа реальных дел — кейсы
  • Школа IT-решений — IT-кейсы

Оформление проекта

Титульный лист: Учебное заведение, тема, ФИО, город, год.

Введение: Проблема, актуальность, цель, задачи.

Теория: 7-10 стр.

Практика: 4-15 стр (в зависимости от класса).

Заключение: Вывод результатов.

Важно! 10 класс — антиплагиат >60%.

Оформление текста

  • Word, А4, Times New Roman 14pt
  • Поля: 30/20/15/20 мм
  • Интервал: полуторный
  • Нумерация: сквозная

Презентация

  • Минимум текста
  • Текст — не визуальное средство
  • Не перегружать слайды
